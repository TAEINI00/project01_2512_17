# 팩트체크로 시작하는 돈 관리 앱 

## 📌 프로젝트 목적
- 10대에서 20대 사이에서 과소비가 유행이다. 그래서 팀원들과 함께 팩트체크로 돈 관리 습관을 잡아주는 ai가 있으면 좋을 것
같다고 의견이 나와 만들게 되었다.

## 🧠 사용 기술
- flutter
- spring boot
- fast api
- my sql

## ⚙ 구현 내용
- 팩폭 ai
- 챌린지방
- 커뮤니티
- 소비습관 분석
