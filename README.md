# 팩트체크로 시작하는 돈 관리 앱 

## 📌 프로젝트 목적
- 10대에서 20대 사이에서 과소비가 심하다는 의견이 나왔다. 그래서 팩트체크로 돈 관리 습관을 잡아주는 ai를 만들게 되었다.
- AI가 팩트 기반 피드백을 제공해 과소비를 인식하게 하고 또래 사용자들과의 챌린지·등급 시스템을 통해 지속 가능한 돈 관리 습관을 형성하도록 돕는 앱

## 🧠 사용 기술
- Flutter
- Spring Boot
- FastAPI
- My SQL

## ⚙ 구현 내용
- 팩폭 ai : 소비 내역을 분석해 ai가 팩트폭력을 하면서 감정적인 위로가 아닌, 객관적인 수치와 비교를 통한 경각심을 유도
- 챌린지방 : 챌린지 달성도에 따라 등급 및 시각적 보상 / 혼자 관리하는 가계부가 아닌, 경쟁과 협력을 통한 습관 형성을 목표로 설계
- 커뮤니티 : 사용자들이 편하게 대화할 수 있는 화면
- 소비습관 분석 : 영수증 데이터를 분석해 사용자의 소비 내역을 분석 / 실제 소비 흐름을 시각적으로 확인
