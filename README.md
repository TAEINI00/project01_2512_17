# 팩트체크로 시작하는 돈 관리 앱 

## 📌 프로젝트 목적
- 10대에서 20대 사이에서 과소비가 심하다는 의견이 나왔다. 그래서 팩트체크로 돈 관리 습관을 잡아주는 ai를 만들게 되었다.

## 🧠 사용 기술
- Flutter
- Spring Boot
- FastAPI
- My SQL

## ⚙ 구현 내용
- 팩폭 ai : 소비 내역을 분석해 ai가 팩트폭력을 하면서 돈 관리 습관을 잡아야함을 상기시켜줌
- 챌린지방 : 사용자들 챌린지방 내에서 등급을 나눠 관리 습관을 잡을 수 있는 동기를 만들어줌
- 커뮤니티 : 사용자들이 편하게 대화할 수 있는 화면
- 소비습관 분석 : 영수증 데이터를 분석해 사용자의 소비 내역을 분석
